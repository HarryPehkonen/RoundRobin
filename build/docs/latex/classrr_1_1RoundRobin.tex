\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class rr\+::\+Round\+Robin$<$ T $>$}
A round-\/robin container that cycles through items in a consistent order. 

This class is designed to provide a round-\/robin scheduling strategy, where each item added to the container is visited in a cyclic manner. It supports movable and copyable types, with special handling for move-\/only types like std\+::unique\+\_\+ptr.
