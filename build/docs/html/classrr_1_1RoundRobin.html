Detailed Description

template<typename T>
class rr::RoundRobin< T >

A round-robin container that cycles through items in a consistent order.

This class is designed to provide a round-robin scheduling strategy, where each item added to the container is visited in a cyclic manner. It supports movable and copyable types, with special handling for move-only types like std::unique_ptr.

Template Parameters
TThe type of items stored in the round-robin container.

Removes the current item from the container.

Should be called only after a successful call to next() or try_next(). The next call to next() or try_next() will return the next item in sequence.

Exceptions
